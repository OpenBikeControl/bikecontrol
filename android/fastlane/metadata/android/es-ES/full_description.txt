Con BikeControl (antes conocido como SwiftControl) puedes controlar tu aplicación de entrenamiento favorita usando tu Zwift® Click, Zwift® Ride, Zwift® Play, Elite Square Smart Frame®, Elite Sterzo Sterzo Smart®, Wahoo Kickr Bike Shift®, mandos Bluetooth y gamepads. Esto es lo que puedes hacer, según tu configuración:

▶ Cambio de marchas virtual
▶ Dirección/giros
▶ Ajustar la intensidad del entrenamiento
▶ Controlar la música en tu dispositivo
▶ Añadir el cambio virtual a aplicaciones de entrenamiento que no lo admiten de forma nativa
▶ Ajustar todo lo relacionado con el cambio virtual: número de marchas, relaciones, etc.
▶ ¿Quieres más? Si puedes hacerlo con teclado, ratón o pantalla táctil, puedes hacerlo con BikeControl.

Uso de la API AccessibilityService
Aviso importante: Esta aplicación utiliza OPCIONALMENTE la API AccessibilityService de Android para permitir el control de las aplicaciones de entrenamiento a través de tus dispositivos Zwift.

¿Por qué AccessibilityService?
▶ Para simular gestos táctiles en tu pantalla que controlan las aplicaciones de entrenamiento.
▶ Para detectar qué ventana de la aplicación de entrenamiento está activa.
▶ Para permitir un control fluido de aplicaciones como MyWhoosh, IndieVelo, Biketerra.com y otras.

Cómo usamos AccessibilityService?
▶ Cuando pulsas los botones de tus dispositivos Zwift Click, Zwift Ride o Zwift Play, BikeControl los traduce en gestos táctiles en ubicaciones específicas de la pantalla.
▶ El servicio monitoriza qué ventana de la aplicación de entrenamiento está activa para garantizar que los gestos se envíen a la aplicación correcta.
▶ NO se accede, recopila ni transmite ningún dato personal a través de este servicio.
▶ El servicio solo realiza las acciones táctiles específicas que configures en la aplicación.

Privacidad y seguridad:
▶ BikeControl solo accede a tu pantalla para realizar los gestos que configures.
▶ No se accede a ninguna otra función de accesibilidad ni a información personal.
▶ Todas las configuraciones de gestos permanecen en tu dispositivo.
▶ Esta aplicación no se conecta a servicios externos para las funciones de accesibilidad.

Esta aplicación no está afiliada ni respaldada por Zwift, Inc., Wahoo ni Elite.

Permisos necesarios
▶ Bluetooth: Para conectarse a tus dispositivos Zwift.
▶ Servicio de accesibilidad (opcional): Para simular gestos táctiles y controlar las aplicaciones de entrenamiento.
▶ Notificaciones: Para que la aplicación se ejecute en segundo plano.
▶ Ubicación (Android 11 y versiones anteriores): Necesaria para el escaneo Bluetooth en versiones anteriores de Android.
